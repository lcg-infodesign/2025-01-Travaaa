Computergrafica per l'Information Design

Project Overview

This project is part of the first assignment for the Information Design course at Politecnico di Milano.
Each student was given a personal dataset and a set of unique rules to determine which rows are valid. Once the valid data is identified, the task is to analyze it and display several statistical measures using p5.js.

Specifically, the project focuses on calculating and representing:
  • the mean of the first column,
  • the standard deviation of the second column,
  • the mode of the third column,
  • the median of the fourth column, and
  • both the mean and standard deviation of the fifth column.
The visualization includes two graphical and two textual representations, plus a fifth element that is freely designed.
Through this exercise, the goal is to explore how numerical data can be translated into meaningful visual forms, making statistics more intuitive and engaging through design and interaction.
